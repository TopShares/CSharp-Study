一：建造者模式


房子： 【单元楼，别墅】

组成部分：【黄沙，瓷砖，水泥，青砖】


黄沙的品牌：【多变】


一个复杂的对象由很多的子部件的组成，子部件是多变，但是子部件组成的流程确实不易变。。


class 单元楼 {
  黄沙
  瓷砖
  水泥
  青砖
}

class 别墅 {
  黄沙
  瓷砖
  水泥
  青砖
}
