
一：Char

    说到Char类型，如果大家学过C语言的话，肯定对这个很熟悉。。。

    在C#中呢，提供了两种方式去定义Char类型

			=> 第一种： 使用char关键字，也就是visual studio 提供给我们的语法糖

			=> 第二种： 使用Char类来定义字符


			char c = 'a';
			Char cc = 'b';

   说到了Char，我们就不得不提Asc码，记住常用字符的asc码

   
   看一下Char类给我们提供了哪些方法？？？

   我们用的比较多的几个方法是那些呢？？？

   1. ToLower() 将指定的字符转换为小写。

   2. ToUpper() 将指定的字符转化为大写。

   3. Parse()   将指定的类型转化为char。

                除了parse方法，我们有一个专门的类，Convert类，也提供了一个叫做ConvertToChar方法。


如果大家对方法看不懂，或者说很好奇，那么怎么办，用ILSpy工具去查看一下源代码

从Lower方法中去追究，我们发现它最终调用的是 Win32方法，但是我们追踪到了一个类：CultureInfo culture

那么这个类就是语言环境类。
		    



二：String
  
    1. 它是一个特殊的引用类型，因为它有值类型的特性。

       《1》参数传参：  对string来说，它是复制，而不是引用。

       《2》我们来看一下String中的方法
          
	     字符串的拼接，我们知道了三种方式：


	         var str = String.Concat("hello", "world");

		 var str2 = "hello" + "world";

		 var str3 = String.Format("{0}{1}", "hello", "world");

        《3》 Equals和Compare方法的区别：

	      Equal    是两种状态，等于或者不等于。

	      Compare  是三种状态，大于，等于，小于。。。。

         《4》 IsNullOrEmpty 判断当前的string是否为empty2016/1/6或者为null。

         《5》 Join           用于将数组或者其他集合类型转换为string。
